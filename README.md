# Spotify Explorer

## Overview

I scraped a bunch of Spotify's auto generated playlists. All the "Sounds of <genre>" and "Sounds of <location>" in addition to a few others.

They are displayed on a website where you can filter and open them in Spotify app.

A great way to discover new music

## Tech

It's a static site with standard Vite / Node build system, deployed on Github pages.

The data is fetched via Ruby scripts, which can be found in scraper/.
